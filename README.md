# Alpaca Settings
add the follow environmental variables using info from Alpaca


### SECRET_KEY
process.env.SECRET_KEY

### API_KEY_ID
key: process.env.API_KEY_ID



# NestJS
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
